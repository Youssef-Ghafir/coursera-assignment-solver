# Coursera Assignment Solver 🎓

A sleek, lightweight Chrome Extension powered by the Google Gemini API to help you extract and solve Coursera assignments directly from your browser. 

## ✨ Features
* **AI Assignment Solver:** Automatically reads the questions on your active Coursera page and generates answers using Gemini 2.5 Flash.
* **Question Extractor:** Neatly extracts all quiz/assignment questions from the page into a clean, formatted interface.
* **One-Click Copy:** Instantly copy the extracted questions to your clipboard.
* **Secure API Key Storage:** Your Gemini API key is saved locally and securely in your browser using Chrome's extension storage.

## 🚀 How to Use

### 1. Get your API Key
You will need a free Google Gemini API key to power the AI.
1. Go to Google AI Studio.
2. Click **Create API Key** and copy the generated key.

### 2. Install the Extension
1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right corner).
4. Click **Load unpacked** and select the folder containing this extension's files.

### 3. Configure and Run
1. Navigate to any Coursera assignment or quiz page.
2. Click the extension icon in your Chrome toolbar.
3. Paste your Gemini API key into the input field and click **Save Key**.
4. Choose your action:
   * Click **Answer the questions !** to get AI-generated solutions.
   * Click **Show extracted questions** to pull the raw text and copy it to your clipboard.
